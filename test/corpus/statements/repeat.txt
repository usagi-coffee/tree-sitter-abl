REPEAT
================================================================================

REPEAT:
  DISPLAY x.
END.

REPEAT WHILE x < 5:
  x = x + 1.
END.

REPEAT UNTIL done:
  DISPLAY done.
END.

main: REPEAT ON STOP UNDO, RETURN ERROR "x" ON ERROR UNDO, THROW:
  LEAVE.
END.
